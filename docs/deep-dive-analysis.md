## The 3 Critical Gaps (Deep Analysis)

### Gap 1: Academic/Scholarly Sources

**Problem**: Tavily finds web content. Firecrawl extracts web pages. But **neither accesses peer-reviewed journals**, conference papers, or academic databases.

**Why This Matters**:
- For "legit good research with good references," we need **citable sources**
- Web blogs aren't citable in academic or professional contexts
- Many technical topics (AI, medicine, science) require peer-reviewed backing

**Top Solutions Discovered**:

| API | Papers | Cost | Best For |
|-----|--------|------|----------|
| **Semantic Scholar API** | 200M+ | FREE | Citation graphs, paper search |
| **OpenAlex API** | 250M+ | FREE | Open alternative to Google Scholar |
| **Consensus** | 220M+ | Freemium | Evidence-based Q&A |
| **Crossref API** | 150M+ | FREE | DOI resolution, metadata |
| **CORE API** | 135M+ | FREE | Open access papers |
| **Scite** | 250M+ | $20/mo | Citation context (supporting/contrasting) |

**Recommendation**: 
- **Primary**: Semantic Scholar API (free, comprehensive, citation data)
- **Secondary**: OpenAlex API (open source, no rate limits)
- **Tertiary**: Consensus (for evidence synthesis)

---

### Gap 2: Citation & Reference Management

**Problem**: We extract content but don't extract **structured citations** or build **reference lists**.

**Why This Matters**:
- Professional research requires proper attribution
- Building on past research requires tracking sources
- Clients/stakeholders expect bibliographies

**Top Solutions**:

| Tool | Function | Cost |
|------|----------|------|
| **Zotero API** | Reference management, tagging | FREE |
| **Crossref API** | DOI metadata, citation formatting | FREE |
| **Citation.js** | Node.js citation formatting | FREE |
| **BibTeX parsers** | Academic format conversion | FREE |

**Recommendation**: 
- Build a **citation extraction module** that:
  1. Extracts author, title, date, URL from scraped content
  2. Queries Crossref for DOI if available
  3. Formats as APA, MLA, or BibTeX
  4. Builds a cumulative reference database

---

### Gap 3: Research Memory & Knowledge Building

**Problem**: Each research query is **stateless**. We don't build a **knowledge base** over time.

**Why This Matters**:
- Research should compound (today's findings inform tomorrow's)
- Entity relationships should be tracked across queries
- Contradictions and updates should be flagged

**Top Solutions**:

| Technology | Function | Cost |
|------------|----------|------|
| **Pinecone/Weaviate** | Vector DB for semantic search | Free tiers |
| **Neo4j** | Knowledge graph for relationships | Free |
| **Obsidian + API** | Markdown knowledge base | Free |
| **LangChain Memory** | Conversation/context memory | Free |

**Recommendation**:
- Add a **vector database** (Pinecone free tier) to store:
  - Scraped content embeddings
  - Research findings
  - Entity profiles
- Add **knowledge graph** tracking:
  - Entity relationships (company → product → competitor)
  - Claim provenance (who said what, when)
  - Confidence scores over time

## New APIs to Add

### 1. Semantic Scholar API (FREE)

**What it does**: Search 200M+ academic papers with citation data

**Key endpoints**:
```
GET https://api.semanticscholar.org/graph/v1/paper/search
  ?query={search query}
  &fields=title,authors,year,citationCount,referenceCount

GET https://api.semanticscholar.org/graph/v1/paper/{paperId}
  ?fields=title,abstract,authors,year,citations,references
```

**Use cases**:
- Find peer-reviewed sources on any topic
- Build citation networks (who cited whom)
- Extract paper metadata for references

**Rate limits**: 100 requests/5min (free tier)

---

### 2. OpenAlex API (FREE)

**What it does**: Open alternative to Google Scholar, 250M+ works

**Key endpoints**:
```
GET https://api.openalex.org/works
  ?search={query}
  &filter=publication_year:2026

GET https://api.openalex.org/authors/{id}
GET https://api.openalex.org/institutions/{id}
```

**Use cases**:
- Author disambiguation
- Institutional affiliation tracking
- Open access paper discovery

**Rate limits**: 100,000 requests/day (very generous)

---

### 3. Crossref API (FREE)

**What it does**: DOI metadata, citation formatting, journal data

**Key endpoints**:
```
GET https://api.crossref.org/works/{doi}
GET https://api.crossref.org/works?query={query}
```

**Use cases**:
- Resolve DOIs to full metadata
- Format citations in any style
- Validate reference accuracy

**Rate limits**: Polite pool (add mailto= parameter)

---

### 4. Consensus API (Freemium)

**What it does**: Evidence-based Q&A over 220M+ scientific papers

**Key endpoints**:
```
POST https://consensus.app/api/search
  Body: { query: "does caffeine improve cognition?" }
```

**Use cases**:
- Quick evidence synthesis
- Find scientific consensus on claims
- Extract key paper findings

**Pricing**: Free tier (limited), Pro at $10/mo
